Db-Mysql-Engine

SHOW ENGINES; 用于查看当前引擎。

Fenerated

1. FEDERATED是其中一个专门针对远程数据库的实现。

  • 在保证两边数据库表结构一致的情况下，用于简单的数据同步。
  • 两边可以是不同的表名
  • 仅支持mysql
  • 不支持事务
  • 不支持表结构的修改
  • CREATE TABLE (……) ENGINE =FEDERATED CONNECTION=’mysql://[name]:[pass]@[location]:[port]/[db-name]/[table-name]’
